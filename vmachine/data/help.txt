  VMachine.exe [--bytecode|--asm] <files...> [-o <dir>]

  --help
  input file type:
    --bytecode (default)
      (.shbyte)
      Virtual machine starts a program.
      Only one file.
    --asm
      (.shasm)
      Creates bytecode files from shizoid assembler.
      Any amount of files.
      -o <dir>
        Directory where .shbyte output files will be placed.
        <dir> - current directory by default.
      // --dupl
      //   Create new files instead of replacing existing files in case of duplicates.
      //   False by default.
